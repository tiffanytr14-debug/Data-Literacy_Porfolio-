# Data-Literacy_Porfolio-
My Data Literacy coursework: **homework (HW01–HW04)** + **prediction challenges (PC01–PC03)**.
Focus: data cleaning, statistical testing, and building predictive models in R.
## Repo map
- `hw/` — homework assignments
- `prediction-challenges/` — model-building challenges + submissions

## Assignments
### Homework
- **HW01** — Exploratory analysis (report included)
- **HW02** — Hypothesis testing + categorical tests
- **HW03** — CLT simulation + permutation test + chi-square test
- **HW04** — *Student-designed* prediction challenge (synthetic visa dataset generator)

### Prediction challenges
- **PC01** — Cars “Deal” classification
- **PC02** — Earnings regression (ranger)
- **PC03** — Hiring classification (rules + random forest)

## Run instructions
Each folder has a `code/` script.
1. Download the dataset for that assignment
2. Put it in the assignment’s `data/raw/` folder
3. Run the script in RStudio

> Datasets are not included (course policy). `data/raw/` is git-ignored by default.
